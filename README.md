# Automotive Inspection App

React Native mobile app for AI-powered automotive parts inspection.

## Tech Stack
- React Native (Expo)
- AsyncStorage
- React Navigation
- Axios

## Screens
- Login
- Camera (capture/upload part image)
- Result (AI analysis + recommendation)
- History (past inspections with stats)

## Setup
1. Clone the repo
2. Run `npm install`
3. Update `src/services/api.js` with your backend IP
4. Run `npx expo start --android`
